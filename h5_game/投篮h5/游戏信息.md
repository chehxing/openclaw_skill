## 游戏控制
| 控制方式 | 功能说明 |
|---------|---------|
| **← →** | 左右移动球员 |
| **长按鼠标** | 蓄力（观察底部指示器） |
| **释放鼠标** | 投篮（力度与蓄力时间成正比） |
| **F键** | 切换全屏模式 |
| **Esc键** | 退出全屏 |

## 游戏机制
1. **蓄力系统**: 长按鼠标0-2秒，力度从5增加到15
2. **力度指示器**: 颜色变化（绿→黄→红）表示力度增加
3. **瞄准系统**: 鼠标移动时显示瞄准线
4. **得分系统**: 篮球进入篮筐得2分
5. **统计系统**: 实时显示得分、尝试次数、命中率
