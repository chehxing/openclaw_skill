# 🏀 投篮游戏 - Basketball Game

一个简单的H5投篮游戏，使用键盘左右箭头键移动球员，鼠标点击投篮。

## 🎮 游戏特点

- **简单易玩**：只需使用箭头键和鼠标即可操作
- **物理真实**：篮球抛物线运动，重力模拟
- **美观界面**：现代渐变设计，流畅动画
- **完整功能**：得分系统、命中率统计、全屏支持
- **响应式**：适配各种屏幕尺寸

## 🚀 快速开始

### 方法一：使用Python HTTP服务器
```bash
# 在项目目录中运行
python3 -m http.server 8080
```
然后在浏览器中打开：http://localhost:8080/basketball_game.html

### 方法二：直接打开
双击 `basketball_game.html` 文件在浏览器中打开

## 🎯 游戏控制

| 控制方式 | 功能说明 |
|---------|---------|
| **← 左箭头** | 球员向左移动 |
| **→ 右箭头** | 球员向右移动 |
| **鼠标点击** | 投出篮球 |
| **F 键** | 切换全屏模式 |
| **Esc 键** | 退出全屏模式 |

## 📊 游戏规则

1. 使用左右箭头键控制球员在篮筐下方移动
2. 点击鼠标投出篮球
3. 篮球会以抛物线轨迹飞向篮筐
4. 篮球进入篮筐得2分
5. 游戏统计：得分、尝试次数、命中率

## 🛠️ 技术实现

### 文件结构
```
basketball_game.html      # 游戏主界面
basketball_game.js        # 游戏逻辑代码
test_game.js             # 功能测试脚本
progress.md              # 开发进度记录
README.md                # 说明文档
```

### 核心技术
- **HTML5 Canvas**：游戏渲染
- **原生JavaScript**：游戏逻辑
- **CSS3**：现代UI设计
- **物理引擎**：自定义重力、碰撞检测

### 关键功能
- `BasketballGame` 类：游戏主逻辑
- `shootBasketball()`：投篮物理计算
- `checkHoopCollision()`：篮筐碰撞检测
- `window.render_game_to_text()`：游戏状态输出（用于测试）
- `window.advanceTime()`：时间步进控制（用于测试）

## 🧪 测试接口

游戏提供了两个测试接口，可用于自动化测试：

```javascript
// 获取当前游戏状态（JSON格式）
const gameState = window.render_game_to_text();

// 手动推进游戏时间（毫秒）
window.advanceTime(100); // 推进100毫秒
```
